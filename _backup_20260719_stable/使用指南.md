# 微机全自动水分测定仪 — 使用指南

> 鹤壁市淇天仪器仪表有限公司 · 版本 20.70
> 适用系统：Windows 7/10/11 · 麒麟 Linux

---

## 第一步：安装 Python（必须）

软件需要 Python 才能运行，请先安装。

### Windows 系统

1. 打开浏览器，访问：**https://www.python.org/downloads/**
2. 点击黄色按钮 **Download Python** 下载安装包
3. 双击下载的安装包
4. ⚠️ **重要**：勾选底部的 **「Add Python to PATH」**（把 Python 加到系统路径）
5. 点击 **Install Now**（立即安装）
6. 等待安装完成，点击 **Close**（关闭）

> 如果下载页面没有黄色按钮，点「Downloads → Windows」，选 3.9 ~ 3.12 之间的版本。

### 麒麟 Linux 系统

系统通常已自带 Python，如果没有，在终端执行：

```bash
sudo apt update
sudo apt install python3 python3-pip
```

---

## 第二步：安装依赖组件（仅首次）

打开 **命令提示符**（Windows）或 **终端**（麒麟），输入以下命令并回车：

```bash
pip3 install pyside2 pyserial pyinstaller
```

等待安装完成（约 2-5 分钟，需要联网）。

> 如果提示 `pip3` 找不到，试试 `pip` 替代 `pip3`。
> 麒麟 ARM64 如果报错，改用系统包安装：
> ```bash
> sudo apt install python3-pyside2 python3-serial
> pip3 install --user pyinstaller
> ```

---

## 第三步：构建程序（仅首次）

### Windows

1. 打开源码所在文件夹
2. 在文件夹地址栏输入 **powershell** 回车
3. 输入以下命令：

```powershell
.\build_windows.ps1
```

### 麒麟 Linux

1. 打开源码所在文件夹
2. 右键 → **在终端中打开**
3. 依次输入：

```bash
chmod +x build_linux.sh
./build_linux.sh
```

构建完成后，程序在 `dist` 文件夹里。

---

## 第四步：启动程序

### Windows

双击 `dist` 文件夹里的 **水分测定仪.exe**

### 麒麟 Linux

双击 `dist` 文件夹里的 **水分测定仪**，或在终端运行：

```bash
./dist/水分测定仪
```

---

## 常见问题

### 程序打不开？

- Windows：确认 Python 安装时勾选了「Add Python to PATH」。如果没勾，重新安装 Python 并勾上。
- 麒麟：确认已给程序执行权限（`chmod +x`）。

### 串口没反应？

```bash
sudo usermod -aG dialout $USER
```

执行后**注销系统重新登录**生效。

### 找不到 data.db？

构建脚本会自动把数据库文件复制到程序旁边，无需手动操作。如果确实没有，把源码里的 `data.db` 复制到 exe 同级目录即可。

### 提示 Python 版本不对？

Python 需要 3.9 ~ 3.12 版本。不要安装 3.13 或更高版本。

---

## 日常使用

首次构建完成后，以后每次使用只需做**第四步**（启动程序），前三步不用重复。
