# 微机全自动水分测定仪 — 麒麟 Linux 构建指南

## 系统要求

- 麒麟 Linux（x86_64 或 ARM64）
- Python 3.8+
- 可访问互联网（首次安装依赖）

## 构建步骤

### 1. 解压源码

```bash
tar xzf water-analyzer-kylin-src.tar.gz
cd water-analyzer-kylin/code
```

### 2. 安装依赖（仅首次）

```bash
pip3 install --user PySide2 pyserial pyinstaller
```

> 如遇权限问题，加 `--user`。麒麟 ARM64 若 PySide2 无预编译包，通过系统包管理器安装：
> ```bash
> sudo apt install python3-pyside2 python3-serial
> pip3 install --user pyinstaller
> ```

### 3. 一键构建

```bash
chmod +x build_linux.sh
./build_linux.sh
```

### 4. 运行

```bash
./dist/水分测定仪
```

## 目录结构

```
water-analyzer-kylin/
├── 使用指南.md          ← 本文件
└── code/
    ├── build_linux.sh   ← 一键构建脚本
    ├── main_app.spec    ← PyInstaller 配置
    ├── main_app.py      ← 主程序入口
    ├── data.db          ← 数据库模板
    └── ...              ← 其余源码模块
```

## 常见问题

**Q: 串口无权限？**
```bash
sudo usermod -aG dialout $USER
# 注销重新登录生效
```

**Q: 程序启动报错找不到 data.db？**
构建脚本已自动将 `data.db` 复制到可执行文件同级目录，无需手动处理。
